# Agent instructions

See [CLAUDE.md](CLAUDE.md) — that's this project's full operating manual (how to run
onboarding, how to edit `calendar-data.js`, how to upgrade the tool and migrate your data
(`npm run update` / `node tools/migrate.js`), the trust contract, the privacy rule for this
being a public template, everything). It's named `CLAUDE.md` for historical/Claude Code
convention reasons, but it's written to apply to any coding agent, and this file exists
so agents that look for `AGENTS.md` by convention (e.g. Codex) find their way to it too.
