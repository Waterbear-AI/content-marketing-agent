# Memory — durable facts & preferences

Cross-topic knowledge that should survive every session, kept **separate from `CLAUDE.md`**
(a stable how-I-work contract) and from `calendar-data.js` (current content *state*). This is
the *who/what/how* of your world — your agent fills it in over time; it ships empty.

**Rules for this file (keep it trustworthy):**
- **Two-strikes:** only record a fact/preference after it's shown up twice. Don't pre-load
  speculation here.
- **Cap ~150 lines.** If it grows past that, prune stale entries or split by topic.
- **Mark inferences** `⚠ assumption:` — same trust contract as everywhere else.
- You are the author of record; you can edit or delete anything here. Your agent proposes,
  you dispose.
